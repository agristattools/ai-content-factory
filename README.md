## 📺 What is AI Content Factory?

AI Content Factory is a **complete YouTube & Facebook automation system** that uses **100% FREE AI tools** to research trending topics, write human-like scripts, generate voiceovers, create professional videos, design thumbnails, and auto-post to social media.

**🌐 Supports: English | Urdu | Hindi | Punjabi**

---

## ✨ Features

### 🎬 Content Creation Pipeline
- **🔍 AI Trend Research** - Finds viral topics from Google Trends & Reddit
- **✍️ AI Script Writer** - Human-style scripts in 4 languages using Llama 3 70B (FREE)
- **🎙️ AI Voiceover** - 300+ natural voices via Microsoft Edge TTS (FREE)
- **🎬 Video Creator** - Professional videos with Ken Burns effects & transitions
- **🎨 Cartoon Studio** - Create animated cartoon series with characters
- **🖼️ Thumbnail Maker** - Click-worthy YouTube thumbnails with AI backgrounds
- **📈 SEO Engine** - High-CTR titles, tags, descriptions

### 📤 Distribution
- **📺 YouTube Uploader** - Auto-upload with SEO metadata
- **📘 Facebook Publisher** - Auto-post to pages
- **⏰ Auto-Scheduler** - Set & forget daily posting

### 🌐 Web Dashboard
- **Streamlit GUI** - Beautiful browser interface
- **Multi-language** - English, Urdu, Hindi, Punjabi
- **One-click** content creation

## 🚀 Quick Start

### Prerequisites
- Python 3.9 or higher
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/ai-content-factory.git
cd ai-content-factory

# Create virtual environment
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Get FREE API keys:
# 1. Groq: https://console.groq.com
# 2. Gemini: https://aistudio.google.com
# 3. YouTube: https://console.cloud.google.com
# 4. Pexels: https://www.pexels.com/api

# Copy and fill .env file
copy .env.example .env
notepad .env  # Add your API keys

# Run!
python main.py
